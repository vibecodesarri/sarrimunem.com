# sarrimunem.com

Personal site — SARRI_OS. One self-contained `index.html`: no build step, no dependencies, no frameworks. Just open it.

## Editing

Everything lives in `index.html`. The copy (about text, project cards, links) is plain HTML near the top of the `<body>` — edit and push.

## Deploying

Hosted on GitHub Pages from the `main` branch. Push to `main` and it ships. The `CNAME` file binds the custom domain `sarrimunem.com`.
